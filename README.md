### DOS命令  
DOS命令是储存在操作系统中的***命令集***，是一种面向磁盘的操作命令，主要包括目录操作类命令、磁盘操作类命令、文件操作类命令和其它命令。能有效地管理在它控制下的*各种软硬件资源*，对它们进行合理的调度。
```
s=0
n=101
for i in range(1,n):	
if(i%2!=0):		
s+=i
print(s)
```
